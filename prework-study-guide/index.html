HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Information in the head element is called Metadata
  • Metadata is data about the webpage that the browser uses to serve webpage users
  • Title element is required by Search Engine Optimization
  • link element connects external resource to the HTML Document
  • meta element defines metadata such as the character, set, desription, keywords, author or viewport
  • script element embeds client side scipt in the HTML Document
  • Semantic HTML elements descibe it's content and the role it plays on the webpage
  • element refers to the entire element from opening tag to closing tag
  • tag refers to what appears inside the angle brackets
  • Section is a samantic element thats defines a section in a document by thematic content
  • id is used in HTML to identify elements and select them

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • main refers to the default branch
  • git add -A: adds all current modifications to branch staging area
  • The staging area informs Git that the changes will be added to the next commit
  • git commit includes message, best if it in present tense
  • git pull origin main:receive branch's modifications to local environment from GitHub Repo on the main branch
  • git push: push can be used to create a new remote branch or add new changes to an existing branch
